  • Remember User ID
    Front-end development
    Project for EverBank.com
    Technology: Sitecore CMS, javascript, html, css, jquery

    This assignment required creating a method to allow online account users to store away their user id's for future sessions. It was determined this feature should only support users of modern browsers. As a result, I opted to take advantage of using html5 local storage.

    The development for this enhancement is complete and the launch of this feature had been postponed for a future date.

  • Mediaplex Retargeting Ads
    Front-end development
    Project for EverBank.com
    Technology: javascript, html, Raphael.js and Animate.css

    These rates do not reflect actual rates of EverBank products. This is a sample and banner only.

    One of the fun projects I had a chance to work on as a developer for EverBank. While this screenshot doesn't do the final design justice, I worked with a designer to create a banner ad that would dynamically pull in and display rates from our rates source.

    My first attempt at creating this banner included using Raphael JS framework. However, since these are retargeting ads, meant to live on various sites, the file needed to be all encompassing and the library proved to be too big. My fall back solution involved recreating these ads to use only css for animation, which I achieved by leveraging the Animate.css library.

  • Downloadable Rates Table
    Front-end development
    Project for EverBank.com
    Technology: Sitecore CMS, javascript, html, css, jquery

    At times, what a client asks for isn't necessarily what is truly needed. I find in these moments, it is our role as a developer to identify these situations and for us to consider responding with an alternate approach. In this instance of the rates table, it was requested of our team to supply rates to a third party vendor. The vendor would require daily refreshed rates, preferrably in a csv file in order for them to feed their web banners.

    I built a tool that provided unlimited access to the most up to date rates. This tool features a download button that converts the list into a downladable csv or txt file.
