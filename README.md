## ğŸš€ Quick Start

### **View the Site**
Simply visit: [https://jannaee.github.io](https://jannaee.github.io)

### **Run Locally**

1. **Clone the repository**
```bash
git clone https://github.com/jannaee/jannaee.github.io.git
cd jannaee.github.io
```

2. **Open with a local server**

**Option A: Using Cursor/VS Code Live Server**
- Install "Live Server" extension
- Right-click `index.html`
- Select "Open with Live Server"

**Option B: Using Python**
```bash
# Python 3
python -m http.server 8000

# Then visit: http://localhost:8000
```

**Option C: Using Node.js**
```bash
npx serve
```

3. **View in browser**
```
http://localhost:8000
```

## ğŸ¨ Design Philosophy

### **Color Palette**
The site uses a custom color scheme derived from my original work-life balance illustration:

```css
--navy-deep: #1a3a52;      /* Primary dark */
--navy-medium: #2d5a7b;    /* Secondary dark */
--cyan: #6DD5ED;           /* Accent blue */
--yellow: #FFD93D;         /* Accent yellow */
--white: #ffffff;          /* Background */
--gray-50: #fafbfc;        /* Light backgrounds */
```

### **Typography**
- **Primary Font**: System font stack (-apple-system, BlinkMacSystemFont, Segoe UI)
- **Responsive Sizing**: clamp() for fluid typography
- **Hierarchy**: Clear heading levels with appropriate sizing

### **Animations**
- Scroll-triggered reveal animations
- Hover micro-interactions
- Smooth transitions (cubic-bezier easing)
- Carousel auto-advance with manual controls
