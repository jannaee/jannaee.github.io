the robotics tinkerer

During the summer of 2014 I had a fabulously wonderful weekend in Amelia Island, Florida while attending a popular javascript conference known as JSCONF.

At this event I built my first node bot, a mini robot controlled by javascript and running from a node server, Johnny-five (a javascript arduino programming framework) and a Sparkcore wireless microcontroller. If you are interested in seeing this project being built out step by step, please visit my blog sickfindings.
